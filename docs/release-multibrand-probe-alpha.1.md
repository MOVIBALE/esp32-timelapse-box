# Multi-brand BLE Probe 0.1.0-alpha.1

This optional Alpha is for community camera hardware validation. It is separate
from stable ESP32 Timelapse Box `v0.1.0`.

## Included

- one ESP32-S3 factory image;
- English and Chinese flash, pairing, one-shot, reconnect, privacy, and recovery guides;
- compatibility/evidence matrix;
- retained Furble MIT license and attribution;
- SHA-256 checksums.

## Not included

- Klipper or Moonraker triggering;
- automatic timelapse capture;
- a compatibility promise for Canon, Fujifilm, Nikon, or Ricoh models;
- Sony support in this image.

Use the stable `v0.1.0` Sony factory image for normal Sony timelapse operation.
Read the bundled guide before flashing and report results using the repository
hardware-test issue form.
